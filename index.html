Portfolio

PORTFOLIO

BREAKFAST CLUB

GitHub | Demo

Full-stack web app built in JavaScript with React, Flux, Express, SQL, CSS/SCSS, and Flexbox for the LEARN Academy final group project. It taps into the Yelp API and pulls in two restaurants each week for users to vote on for the Friday morning Breakfast Club tech meetup. Users can sign up, log in, receive email notifications, vote, RSVP, and talk to each other in real time through a message board. I absolutely loved working on this project; it was a defining experience that put theory into practice for me and leveled me up from student to developer.

DRUMKIT

GitHub | Demo

The first project of Wes Bos’ 30 day vanilla JavaScript challenge. This interactive frontend web app – built purely in ES6 – transforms your keyboard into a drumkit by linking audio snippets to keydown events. As a new developer, building this project was an outside-the-box enlightening experience as to what’s possible with JavaScript and how highly interactive apps can be created with very simple code. Turn up your volume and try it out!

5-DAY WEATHER FORECAST

GitHub | Demo

Wondering about the temperature, pressure or humidity in the next 5 days? This full-stack app draws from the OpenWeatherMap API to display the forecast of any city you type in. More importantly, it gave me practice with React, Redux, ReduxPromise, practical usage of ES6 syntax, and knowledge of the various pitfalls to avoid.

Credit: Modern React with Redux Course

CSS TRANSITIONS

GitHub | Demo

This trippy project reinforced my knowledge of HTML event attributes and CSS animations. Using X and Y, offset, and mousemove, the styles dynamically update in accordance with the location of the cursor. The greatest challenge involved the need to offset the values of X and Y in the children elements so these values would be uniform across the page.

Credit: Wes Bos’ 30 day vanilla JavaScript challenge

MS PAINT IN JAVASCRIPT

GitHub | Demo

The 1985 classic, reimagined in JavaScript with HTML5 Canvas.
